The Life of a Radar

Hi there, my parents and friends call me Ryan Bigg, but you may know me as "Radar". Either is fine.

I'm a technical writer for Ruby / Rails based in Australia. I have written a Rails book with Yehuda Katz called Rails 3 in Action, due out this month.

I won a Ruby Hero award in 2011 for my work on documentation for Rails. I keep some guides in my guides repository on GitHub. As part of this documentation, I've worked on the Asset Pipeline, Active Record Query Interface, Configuring Rails Applications and the Rails Initialization Internals guides.

If you like what we do, support us by buying a copy of Rails 3 in Action. You can also subscribe to this blog.

Occasionally I find some time to write interesting blog posts such as the ones you see below.

I also have done quite a lot of open-source work and I tweet. Find out more about me.

Oh, and I once beat DHH in an argument.
